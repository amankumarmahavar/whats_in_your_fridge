## Introduction  

**What's in My Fridge** is a **voice-activated recipe app** that helps users find the best recipes based on the ingredients they have. By leveraging **AI-powered processing** and an extensive recipe database, the app makes meal planning effortless and intuitive.  

## Features  

âœ… **Voice Recognition**: Speak your available ingredients, and the app will generate relevant recipes.

âœ… **Detailed Recipe Information**: Provides ingredients, preparation steps, dish types (**lunch, dinner, snacks**), and more.  
âœ… **Dietary Preferences**: Supports filters for **primal, gluten-free, vegan**, and other dietary needs.  
âœ… **Serving Sizes**: Displays the number of servings per recipe.  
âœ… **Intuitive UI**: A clean, user-friendly design enhances the recipe discovery experience.  

## Technologies Used  

- **Flutter** - Cross-platform UI framework for a seamless mobile experience.  
- **Gemini AI Engine** - AI-powered processing for voice recognition and ingredient interpretation.  
- **Spoonacular API** - Recipe retrieval with detailed nutritional and preparation information.  

## Installation

**Clone the Repository:**

```bash
  git clone https://github.com/amankumarmahavar/whats_in_your_fridge.git
  cd whats_in_your_fridge
```
    
**Check for flutter setup and connected devices**
  ```bash
  flutter doctor
```  

**Get all the dependencies using**
```bash
  flutter pub get
```

**Run the app**
```bash
  flutter run
```
